# Projekt Web Scraper

## Opis
Rozproszona aplikacja do scrapowania stron internetowych, która wyciąga e-maile, adresy i inne dane, zapisując je w MongoDB. Stworzona z użyciem Flask, wielowątkowości i asyncio.
